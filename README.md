# Doruk Söve - Premium Architectural Wall Elements

This repository contains a modern, minimal Next.js (App Router) site starter for a Turkish architectural wall decor company.

Key features:
- Glassmorphism UI with large typography and whitespace
- Hero-driven homepage, product gallery, product detail, about, and contact pages
- Simple in-browser CMS (localStorage) for product management
- Tailwind CSS, Framer Motion for subtle animations
- Responsive, fast, SEO-friendly structure
